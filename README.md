### About

_QuakeX is a comprehensive earthquake data visualization and analytics platform that transforms complex seismic information into accessible, actionable insights. Built with modern web technologies, QuakeX provides real-time earthquake monitoring, interactive mapping, and powerful SQL-based analytics through Apache Drill integration._

### Development environment

- OS : [Windows 11](https://www.microsoft.com/en-us/software-download/windows11)
- Code Editor : [VSCode](https://code.visualstudio.com/download)
- API Testing : [Postman](https://www.postman.com/downloads/)
- Containerization : [Docker](https://www.docker.com/products/docker-desktop/)
- Frontend Framework : [Vue.js](https://vuejs.org/guide/quick-start)
- Backend Framework : [Node.js](https://nodejs.org/en/download/current)
- Database: [Apache Drill](https://drill.apache.org/download/)
- APIs & Data Sources
  - [USGS Earthquake API](https://earthquake.usgs.gov/fdsnws/event/1/) - Real-time seismic data
  - [REST Countries API](https://restcountries.com/) - Comprehensive country information
  - [Open-Meteo Elevation API](https://open-meteo.com/en/docs/elevation-api) - Elevation data
  - [Open-Meteo Weather API](https://open-meteo.com/en/docs/historical-weather-api) - Historical weather data
